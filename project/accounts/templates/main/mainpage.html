Django MTV 정리

🦁MTV란?

모델 (Model), 템플릿 (Template), 뷰 (View)의 약자

  • 모델:

    데이터베이스와의 상호 작용을 관리. 모델은 데이터에 접근하고 조작하는 데 사용

  • 템플릿:

    사용자에게 보여지는 HTML 페이지의 디자인

  • 뷰:

    웹 애플리케이션의 비즈니스 로직을 처리하는 부분. 사용자에게 요청을 받아서 모델에서 데이터를 가져오고, 그것을 템플릿에 전달하여 사용자에게 응답 생성
